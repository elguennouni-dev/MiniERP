v1.0.0

Welcome to ABDO-BOOT!

Your modern Java web framework for building robust and scalable applications with ease.

Simple & Intuitive

ABDO-BOOT offers a clean API that makes web development straightforward and efficient.

Lightweight

Designed to be minimal with no unnecessary dependencies, ensuring fast startup times.

Highly Extensible

Built with extensibility in mind, allowing you to integrate with your favorite libraries.

Getting Started

Add ABDO-BOOT to your project and start building:

// Initialize the ABDO-BOOT application
AbdoBoot app = new AbdoBoot();

// Configure your routes
app.get("/hello", (req, res) -> {
    res.html("Hello, World!");
});

// Start the server
app.start(8080);
